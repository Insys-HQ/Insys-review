💰 크레딧 절감을 위한 초기 세팅법
🎯
Claude는 토큰(문자량) 기준으로 세션이 소모됩니다. 아래 초기 설정을 한 번만 해두면 같은 작업을 더 적은 토큰으로 처리할 수 있어 세션을 훨씬 오래 쓸 수 있습니다.
SET 1 사용자 환경설정 (User Preferences) 등록 최우선 설정
Claude가 매 대화마다 내 정보를 새로 물어보지 않도록, 나에 대한 기본 정보를 한 번만 등록해 두세요. 반복 설명 = 토큰 낭비입니다.
화면 우측 상단 프로필 아이콘 → Settings → Profile → User Preferences 입력
📝 입력 예시 (복사 후 본인에 맞게 수정)
나는 한국어로 대화한다. 직책: [직책명], 회사: [회사명] 업무: [주요 업무 설명] 답변은 항상 간결하게 핵심만 요약. 불필요한 서론/마무리 인사 생략. 코드는 항상 완성본으로 제공.
📉 효과: 매 대화 도입부 반복 설명 제거 → 대화당 200~500 토큰 절감
SET 2 응답 스타일(Style) — 간결 모드 설정 토큰 직결
Claude 기본값은 친절하고 긴 답변입니다. Style 설정으로 간결 모드를 기본값으로 바꾸면 토큰 소모가 크게 줄어듭니다.
Settings → Profile → Communication Style → Concise 선택
❌ 기본 (긴 답변)
"안녕하세요! 좋은 질문이시네요. 이 문제에 대해 여러 관점에서 상세히 설명드리겠습니다. 우선 첫 번째로…"
→
✅ Concise (간결)
"결론: [핵심]. 이유: [한 줄 설명]."
📉 효과: 답변 길이 30~50% 단축 → 체감 세션 최대 2배 연장
SET 3 프로젝트(Project) 기능으로 반복 컨텍스트 제거 업무 효율 핵심
매번 "우리 회사는 이런 곳이고…"라는 설명을 반복하는 건 큰 토큰 낭비입니다. 프로젝트에 배경 정보를 한 번만 등록하면 해당 프로젝트의 모든 대화에 자동 적용됩니다.
왼쪽 사이드바 → Projects → New Project → Project Instructions 입력
📌 Project Instructions 예시
## 회사 정보 회사명: [회사명] | 업종: [업종] 주요 업무: [업무 설명] ## 답변 규칙 - 답변은 항상 한국어 - 결론 먼저, 부연 나중 - 표와 목록 적극 활용 - 코드는 완성본으로 제공
📉 효과: 도입부 설명 완전 제거 → 업무 유형별 프로젝트 분리 강력 권장
SET 4 불필요한 기능 비활성화 즉시 적용
사용하지 않는 기능이 켜져 있으면 토큰이 추가 소모됩니다. 필요할 때만 켜는 습관이 중요합니다.
🔍
웹 검색 (Web Search) — 필요시만 ON
검색 결과 자체가 대량 토큰을 소모합니다. 최신 정보가 필요 없는 업무엔 OFF 유지.
입력창 하단 🔍 아이콘 → 클릭하여 OFF
🧠
메모리 기능 (Memory) — ON 권장
Claude가 내 작업 패턴·선호를 기억해 반복 설명이 불필요해집니다. 토큰 절감 효과가 큽니다.
Settings → Memory → ON
📎
파일 첨부 최소화
대용량 파일 전체 첨부 대신 필요한 부분만 복사해서 붙여넣기. 파일 1개 = 수천 토큰 소모 가능.
필요한 내용만 텍스트로 붙여넣기 권장
🎨
아티팩트 (Artifacts) — 필요시만
코드·문서 아티팩트는 일반 답변보다 토큰 소모가 많습니다. 단순 질의응답엔 OFF.
Settings → Features → Artifacts
SET 5 대화 습관으로 토큰 아끼기 매일 실천
✅ DO — 토큰 절약
  • 질문을 한 번에 명확하게 작성
  • 이미 답변된 내용은 "위 내용 기준으로" 참조
  • 업무별로 별도 대화(채팅) 사용
  • 긴 대화가 끝나면 새 대화 시작
  • 형식 미리 지정: "표로 정리해줘"
  • "3줄 요약", "핵심만" 등 간결 지시
❌ DON'T — 토큰 낭비
  • 하나의 대화를 무한정 이어가기
  • 같은 내용을 여러 번 재설명
  • 대용량 파일 전체 반복 첨부
  • 애매한 질문으로 반복 수정 요청
  • "더 자세히", "더 길게" 무한 확장
  • 사용하지 않는 기능을 켜둔 채 방치

❓
Q. 설치 후 바가 안 보입니다.
Claude.ai 탭을 완전히 새로고침(F5 또는 Ctrl+R)해 보세요. 그래도 안 보이면 확장프로그램 관리 페이지에서 활성화 상태를 확인하세요.
❓
Q. 개인정보는 안전한가요?
이 확장은 API 키나 대화 내용을 수집하지 않습니다. Claude.ai 로그인 세션 쿠키를 이용해 사용 통계만 조회합니다. 모든 데이터는 로컬에 저장됩니다.
❓
Q. Messages left가 N/A로 표시됩니다.
Claude Pro/Max는 토큰 기반 제한이라 정확한 메시지 수를 계산하기 어렵습니다. N/A는 정상입니다. Session %와 Reset in 시간만 참고하면 됩니다.
💡
Tip. 세션을 효율적으로 쓰려면?
세션이 70% 이상이면 무거운 작업(긴 문서 분석, 코드 대량 생성)은 잠시 후 새 세션에서 하는 것이 좋습니다. 업무별로 다른 대화를 사용하면 컨텍스트 낭비를 줄일 수 있습니다.
