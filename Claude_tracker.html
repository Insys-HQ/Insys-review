Chrome 브라우저 설치 방법
1
Chrome 웹스토어 접속
Chrome 브라우저를 열고 주소창에 아래를 입력하거나, 위 바로가기 링크를 클릭합니다.

chrome.google.com/webstore
또는 Chrome 메뉴에서:
⋮ 메뉴 → 도구 더보기 → 확장 프로그램 → Chrome 웹 스토어 열기
2
확장프로그램 검색
웹스토어 검색창에 다음과 같이 입력합니다:

🔍 Claude Usage Tracker
검색 결과에서 lugia19가 만든 "Claude Usage Tracker"를 선택합니다.
3
Chrome에 추가 클릭
확장프로그램 상세 페이지에서 파란색 버튼을 클릭합니다:
Chrome에 추가
팝업이 나타나면:
확장 프로그램 추가
를 클릭합니다. 설치가 자동으로 진행됩니다.
4

이미 Claude.ai가 열려 있다면 F5 또는 Ctrl+R로 새로고침합니다.

채팅 입력창 위쪽에 세션 바가 자동으로 나타납니다:

Firefox 브라우저 설치 방법
1
Firefox 부가기능 페이지 접속
위 Firefox 부가기능 바로가기 링크를 클릭하거나, Firefox 메뉴에서:
☰ 메뉴 → 부가기능 및 테마 → 검색창에 Claude Usage Tracker
2

부가기능 페이지에서:

권한 확인 팝업이 나타나면 추가를 클릭합니다.
3
Claude.ai 새로고침
Claude.ai 탭에서 F5로 새로고침하면 세션 바가 나타납니다.
표시 정보 이해하기
📊
Session: XX%
현재 5시간 롤링 세션에서 사용한 비율. 100%가 되면 5시간 세션 제한에 도달.
⏱️
Reset in: Xh Xm
현재 세션이 초기화되기까지 남은 시간. 이 시간이 지나면 세션 제한이 풀림.
💬
Messages left: N/A
남은 메시지 수. N/A는 토큰 기반 제한으로 정확한 수를 알 수 없을 때 표시됨.
🔴
빨간 마커(▼)
현재 세션 위치를 나타내는 마커. 오른쪽으로 갈수록 한도에 가까워짐.
⚠️
색상 변화: 바의 색상이 파란색→주황색(70%)→빨간색(90%)으로 변합니다. 빨간색에 가까우면 곧 세션이 제한될 수 있습니다.

자주 묻는 질문
❓
Q. 설치 후 바가 안 보입니다.
Claude.ai 탭을 완전히 새로고침(F5 또는 Ctrl+R)해 보세요. 그래도 안 보이면 확장프로그램 관리 페이지에서 활성화 상태를 확인하세요.
❓
Q. 개인정보는 안전한가요?
이 확장은 API 키나 대화 내용을 수집하지 않습니다. Claude.ai 로그인 세션 쿠키를 이용해 사용 통계만 조회합니다. 모든 데이터는 로컬에 저장됩니다.
❓
Q. Messages left가 N/A로 표시됩니다.
Claude Pro/Max는 토큰 기반 제한이라 정확한 메시지 수를 계산하기 어렵습니다. N/A는 정상입니다. Session %와 Reset in 시간만 참고하면 됩니다.
💡
Tip. 세션을 효율적으로 쓰려면?
세션이 70% 이상이면 무거운 작업(긴 문서 분석, 코드 대량 생성)은 잠시 후 새 세션에서 하는 것이 좋습니다. 업무별로 다른 대화를 사용하면 컨텍스트 낭비를 줄일 수 있습니다.
