📋 메뉴 검토 기준

각 모듈에 대해 아래 3가지 기준으로 판단해 주세요
✅ 채택 — 필요 · 지금 당장 써야 함
  • 현재 엑셀로 하고 있는 업무
  • 매달 반복되는 급여·회계·출퇴근
  • 법적 의무 (4대보험·ISO·HACCP)
  • 경영진이 자주 확인하는 지표
⏸️ 보류 — 필요하지만 나중에
  • 당장 급하지 않은 기능
  • 외부 연동이 필요한 복잡한 기능
  • 운영 안정화 후 추가할 기능
  • 예산·인력이 확보된 후 검토
❌ 제거 — 불필요 · 삭제
  • insys 업종과 무관한 기능
  • 동일 기능이 이미 다른 모듈에 있음
  • 사용 빈도가 연 1회 미만
  • 비용 대비 효과가 없는 기능
💡 판단이 어려우면 '보류'로 표시하세요. 개발팀이 비용과 일정을 고려해 우선순위를 재조정합니다.
