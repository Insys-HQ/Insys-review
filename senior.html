핵심

불법파견 방지 설계 원칙

가장 중요한 설계 기준

›
✅ 플랫폼 역할 한정 매칭·정산·평판 관리만 제공.
업무 지휘·감독·평가는 반드시 사업장이 직접 수행.
약관에 "플랫폼은 고용관계 당사자 아님" 명시 필수.
8단계 순서대로 완료 시 합법 운영 가능. 총 소요 약 2~3개월.
STEP
1

법인 정관 목적 추가

직업소개업·소프트웨어업 추가 / 법무사 의뢰 30~50만원 / 1~2주

STEP
2

직업상담사 자격자 확보

등록 필수 요건 / 경영·심리·사회복지 학과 졸업자 대체 가능 여부 사전 확인

STEP
3

사무실 확보 (전용 10㎡+)

창원 HQ 13층 또는 7층 공간 지정 가능

STEP
4

창원시청 유료직업소개업 등록

일자리경제과 방문 / 7~14일 처리 / 무료
⚠️ 이 등록 없이 영업 = 형사처벌

STEP
5

사업자등록 업종 추가

홈택스 / 749100(직업소개) + 722000(소프트웨어) / 당일

STEP
6

방통위 위치정보사업 신고

GPS 출퇴근 필수 조건 / 처리 약 30일
⚠️ 앱 출시 35일 전 신청

STEP
7

개인정보처리방침 수립 + CPO 지정

경력·자격증 = 민감정보 / 별도 동의 항목 추가 / 2~4주

STEP
8

전자계약 + 법무 최종 검토

카카오 전자계약 API 연동 / 약관 법무법인 검수 / 3~4주

인시스 기존 인프라 최대 재활용

  • 결제

  • AI
    Claude API (RAG) — R&D센터 연구과제 연계
  • 서버
    AWS (기존 계정) — 초기 인프라 비용 없음

이것이 핵심 차별화

›

  • 📞
    전화 CS 채널: 60대는 채팅보다 전화 선호 → 전화 상담 필수
  • 🔐
    간편 가입: 전화 본인인증 중심 / 소셜 로그인 보조 / 단계 최소화
  • 📜
    경력 인증: 자격증 사진 촬영 → OCR 인식 → 국가자격증 API 검증 → 인증 배지
  • 📍
    QR 출퇴근 우선: 지하공장 GPS 불량 대비 / 버튼 크기 최대화
  • 🌙
    다크모드·고대비: 시력 약한 고령자 배려

인시스 기존 인프라 연계

  • 💼
