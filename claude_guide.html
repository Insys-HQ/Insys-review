중간에 파일로 저장하면 수정할 때마다 다시 변환해야 하므로 비효율적입니다.

STEP
1
📝 마크다운 초안 작성 요청

Claude에게 문서 종류와 핵심 내용을 알려주고 마크다운 형식으로 초안을 생성합니다.
이 단계에서 완성도를 따지지 말고 빠르게 뼈대만 잡으세요.

아래 내용으로 [보고서/검토서/제안서]를 마크다운 형식으로 초안 작성해줘.
지금은 완성본이 아니라 구조와 뼈대만 잡아줘.

주제: [주제 입력]
포함할 내용: [핵심 항목 나열]
분량 목표: A4 약 [1~2]페이지
STEP
2
✏️ 초안 검토 후 1차 수정 요청

Claude가 생성한 초안을 읽고 빠진 내용, 잘못된 내용, 어색한 부분을 찾아 수정을 요청합니다.
수정 요청은 구체적으로 — "다시 써줘"보다 "2번 항목에 ~를 추가해줘"가 훨씬 효율적입니다.

수정 요청 예시:
"2번 항목(현황 분석)에 구체적인 수치를 추가해줘"
"결론 부분을 좀 더 강하고 설득력 있게 바꿔줘"
"전체 분량이 너무 길어. A4 1페이지로 줄여줘"
"서두 문단을 더 공식적인 문체로 바꿔줘"
"표 형식으로 정리해줘 — 항목, 내용, 비고 3열로"
STEP
3
🔁 만족할 때까지 반복 수정

같은 대화창에서 계속 수정 요청을 이어갑니다. 수정 횟수에 제한은 없습니다.
단, 수정 전에 마음에 드는 버전을 복사해두는 습관이 중요합니다.

💡
중간 저장 팁 수정을 요청하기 전, Claude 답변 아래 📋 복사 아이콘을 클릭해서 현재 버전을 메모장에 임시 저장해두세요.
수정 후 마음에 안 들면 저장해둔 버전으로 되돌아갈 수 있습니다. 
반복 수정 요청 패턴:
"위 내용에서 [특정 부분]만 [이렇게] 바꿔줘. 나머지는 그대로 유지해줘"
"[항목명] 섹션을 통째로 다시 써줘. 핵심 메시지는 [내용]이야"
"전체적으로 문장을 더 간결하게 다듬어줘"
"빠진 내용: [추가 내용]. 어울리는 위치에 넣어줘"
STEP
4
✅ 최종 확정 — 마크다운 전체 출력

수정이 완료되면 최종 완성본을 깔끔하게 한 번에 출력해달라고 요청합니다.

"지금까지 수정한 내용을 반영해서 최종 완성본을 마크다운으로 깔끔하게 처음부터 끝까지 출력해줘."

→ 출력된 내용이 최종본입니다. 이 내용을 복사해서 다음 단계(파일 변환)로 이동하세요.

STEP
5

PDF · Word · Excel 파일 저장 방법

⚠️
Claude는 직접 PDF를 다운로드해주지 않습니다. 아래 3가지 방법 중 선택하세요.
1

브라우저 인쇄 기능 ⭐ 가장 간단

  1. Claude 답변이 완성되면 키보드 Ctrl + P (Windows) / Cmd + P (Mac)
  2. 인쇄 창에서 대상 프린터: "PDF로 저장" 선택
  3. "저장" 클릭 → PDF 파일 완성
2

한글(HWP) 또는 Word로 저장

  1. Claude 답변 전체 복사 (Ctrl+A → Ctrl+C)
  2. 한글 또는 Word에 붙여넣기
  3. 서식 정리 후 PDF로 내보내기
3

Claude에게 직접 파일 요청

"위 내용을 Word 문서(.docx) 파일로 만들어줘"
또는
"위 내용을 다운로드 가능한 파일로 만들어줘"

→ Claude가 파일을 생성하여 다운로드 링크를 제공합니다.
