Part 1
Google Tasks란?

Google Tasks는 구글이 제공하는 무료 할일(Todo) 관리 앱입니다. Gmail, 구글 캘린더와 완벽하게 연동되며, 마감일을 설정하면 캘린더에 자동으로 표시됩니다.

✅ Google Tasks 역할

  • 오늘 해야 할 일 목록 관리
  • 업무별 체크리스트
  • 프로젝트 하위 할일 분류
  • 완료 체크 및 이력 관리

📅 캘린더 역할

  • 회의·약속 등 시간 기반 일정
  • 반복 일정 설정
  • 팀원과 일정 공유
  • Tasks 마감일 시각화
💡
핵심 포인트Tasks에서 마감일을 설정하면, 구글 캘린더에서 해당 날짜에 체크박스 아이콘으로 자동 표시됩니다. 두 앱을 함께 쓰면 일정과 할일을 한눈에 볼 수 있어요!

💻 PC에서 사용하기 (설치 불필요)

1
웹 브라우저로 접속

크롬, 엣지, 사파리 등 브라우저 주소창에 입력:

🔗 tasks.google.com  → 클릭하여 바로 열기

구글 계정으로 로그인하면 바로 사용 가능합니다.

2
Gmail 우측 사이드바에서 열기

Gmail(mail.google.com)에 접속 후, 화면 오른쪽 사이드바에서 체크박스 아이콘을 클릭하면 Tasks 패널이 열립니다.

📧 Gmail 열기
→
우측 사이드바
→
✅ Tasks 아이콘 클릭
3
구글 캘린더 사이드바에서 열기

calendar.google.com 접속 후, 오른쪽 사이드바의 Tasks 아이콘을 클릭하면 캘린더를 보면서 할일을 동시에 관리할 수 있습니다.

✏️ 첫 번째 할 일 추가하기

1
할일 추가 버튼 탭/클릭

화면 하단 또는 우측의 ＋ 할일 추가 버튼을 누릅니다.

2
제목 입력

할일 내용을 간략하게 입력합니다. (예: 급여명세서 앱 개발 완료)

3
날짜/시간 설정

달력 아이콘을 눌러 마감일을 선택합니다. 날짜 설정 시 캘린더에 자동 반영됩니다.

4
목록(리스트) 선택

기본 내 Tasks 또는 새 목록을 만들어 분류할 수 있습니다. (예: 연구소 업무, 경영지원, 개발 등)

Part 3
Google Tasks ↔ 캘린더 연동 확인

Tasks와 구글 캘린더는 같은 구글 계정으로 로그인하면 자동 연동됩니다. 별도 설정 없이 마감일이 있는 할일이 캘린더에 나타납니다.

1
캘린더에서 Tasks 표시 확인

calendar.google.com 접속 → 좌측 하단의 "내 캘린더" 섹션에서 Tasks에 체크가 되어 있는지 확인하세요.

⚠️
Tasks 체크박스가 해제되어 있으면 캘린더에 할일이 보이지 않습니다. 반드시 체크 상태로 유지하세요!
2
Tasks 항목 캘린더에서 확인

마감일을 설정한 Tasks 항목은 해당 날짜에 파란 체크박스 아이콘으로 표시됩니다. 캘린더에서 직접 완료 체크도 가능합니다.

3
모바일 앱에서 동기화 확인

Google Tasks 앱과 Google 캘린더 앱을 같은 계정으로 로그인하면, PC에서 추가한 항목이 모바일에서도 즉시 반영됩니다.

Part 4
Claude에 Google 캘린더 커넥터 설정하기

Claude에 Google 캘린더 커넥터를 연결하면, 대화창에서 말로 일정을 추가·조회·수정할 수 있습니다. 아래 순서대로 따라하세요.

1
claude.ai 접속 및 로그인
🔗 claude.ai  → 클릭하여 바로 열기

Anthropic 계정 또는 구글 계정으로 로그인합니다.

2
대화창 하단 도구 아이콘 클릭

메시지 입력창 왼쪽의 🔧 도구(Tools) 아이콘 또는 ⊕ 아이콘을 클릭합니다.

💬 대화창 열기
→
🔧 하단 도구 아이콘
→
앱 연결 메뉴
3
"Google 캘린더" 찾기

앱 목록에서 Google Calendar를 찾아 [Connect] 또는 [연결] 버튼을 클릭합니다.

🔍
검색창에 "Google" 또는 "Calendar"를 입력하면 빠르게 찾을 수 있어요.
4
구글 계정 권한 허용

팝업 창이 열리면 사용할 구글 계정을 선택하고, 캘린더 접근 권한을 [허용(Allow)]합니다.

🔒
보안 안내: Claude는 허용된 캘린더 읽기·쓰기 권한만 사용합니다. 다른 구글 서비스(드라이브, 이메일 등)는 별도 허용 없이 접근하지 않습니다.
5
연결 완료 확인

도구 아이콘 옆에 "Google Calendar 연결됨" 표시가 나타나면 완료입니다! 이제 대화창에서 말로 일정을 관리할 수 있습니다. ✅

②
업무 중 — 떠오르는 할일은 즉시 Tasks에
