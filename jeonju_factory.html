🚀 빠른 처리를 위한 실전 전략

📌 원스톱 처리 창구 활용

  • FEMIS(femis.go.kr): 공장설립 온라인 원스톱 처리. 입주계약~공장등록까지 온라인 신청
  • 기업지원플러스(G4B): 사업자등록+4대보험 동시 변경 처리
  • 정부24(gov.kr): 대기·폐수·소음 배출시설 신고 온라인 접수
  • 공단관리사무소 방문 시 입주계약+공장등록 동시에 담당자 배정 요청

📌 환경 인허가 단축 전략

  • 대기배출시설 신고 시 수질·소음 의제처리 신청으로 3개 신고를 1회 접수로 처리
  • 폐수 전량 위탁처리 시 허가 → 신고로 간소화 가능여부 담당자 확인
  • 위탁처리업체 계약서를 사전에 준비해 서류 누락 방지
  • 집진장치 설치 계획서 첨부 시 대기 신고 처리기간 단축

📌 선행 준비 서류 목록

  • 법인등기부등본 (최신, 발급일 3개월 이내)
  • 사업자등록증 사본
  • 공장 배치도 / 평면도 (제조시설 위치 표시)
  • 폐수 위탁처리업체 허가증 사본 + 계약서
  • 건축물대장 (신 공장 건물)
  • 법인인감 + 인감증명서

📌 기관별 사전 유선 상담 권장

  • 공단관리사무소: 입주계약 명의변경 vs 신규계약 여부 확인
  • 전주시청 환경과: 폐수배출시설 신고/허가 구분 사전 안내 요청
  • 전주시청 산업과: 공장등록 필요 서류 사전 확인
  • 방문 전 담당자 이름과 직통 전화번호 확보 → 처리 속도 개선

📌 처리 순서 황금 원칙

  • ① 입주계약 → ② 환경 신고(폐수 등) → ③ 공장등록 순서 반드시 준수
  • 환경 허가 없이 공장등록 불가. 공장등록 없이 생산 활동 제재 대상
  • 사업자등록 변경과 4대보험은 이전 당일~3일 내 처리가 이상적
  • 전기·소방 안전관리자는 공장 가동 전에 반드시 선임 완료

📌 비전검사기 조립 공간 관련

  • 비전검사기 조립 → 제조업 범위 내, 공장등록 업종에 조립/가공 업종 포함 확인
  • 기존 산단 입주 업종과 조립·가공 업종의 적합성 공단관리사무소 확인 필요
  • 세척장 내 공간 분리 시 용도별 구획이 배치도에 명확히 표현되어야 함
  • 연구소(AI R&D센터) 파인 연계 조립 공간은 연구개발 목적으로 공장등록과 별도 관리 가능
